# 🍔 QuickBite – Full Stack Food Delivery Application

QuickBite is a full-stack food delivery application built to simulate a real-world ordering platform.  
It allows users to browse restaurants, view menus, place orders, and manage authentication securely.

The application is built using a React frontend, Node.js/Express backend, and PostgreSQL database, following clean architecture and modular design principles.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login (JWT Authentication)
- Browse Restaurants & Menus
- Add Items to Cart
- Place Orders
- View Order History

### 🛠 Admin Features
- Manage Restaurants
- Add / Update Menu Items
- Track Orders

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML5, CSS3
- Axios for API communication

### Backend
- Node.js
- Express.js
- RESTful API architecture
- JWT Authentication

### Database
- PostgreSQL
- Normalized schema design
- Query optimization & indexing

### Tools
- Git & GitHub
- Docker (backend containerization)
- Postman (API testing)

---

## 🏗 System Architecture

- Full-stack client-server architecture
- REST API-based communication
- Layered backend structure (Routes → Controllers → Services → DB)
- Secure authentication using JWT middleware
- Structured relational schema for transactional data integrity
